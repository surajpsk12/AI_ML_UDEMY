# 🌸 Iris Flower Species Predictor

An interactive Machine Learning web application built using Streamlit and Scikit-learn to predict Iris flower species based on flower measurements.

---

## 🚀 Live Features

- 🌿 Predict Iris flower species
- 🎛️ Interactive Streamlit sliders
- 🤖 Random Forest ML Model
- 📊 Prediction confidence visualization
- 📂 Dataset preview
- ⚡ Fast and lightweight UI

---

## 🧠 Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

## 📂 Dataset Used

This project uses the famous Iris Dataset from Scikit-learn.

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target:
- Setosa
- Versicolor
- Virginica

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/surajpsk12/Iris-Flower-Predictor.git
```

Move into the project folder:

```bash
cd Iris-Flower-Predictor
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```

---

## 📊 Machine Learning Model

Model Used:
- Random Forest Classifier

The model is trained on the Iris dataset using Scikit-learn.
